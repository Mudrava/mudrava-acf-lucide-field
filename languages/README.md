# Translations

This directory is used by WordPress.org to store automatically generated translation files.

When your plugin is hosted on WordPress.org, translations are managed through the [WordPress.org translation platform](https://translate.wordpress.org/).

You do not need to manually add translation files here. WordPress will automatically load translations from this directory as needed.

## Text Domain

This plugin uses the text domain: `lucide-acf-field-free`

## Translating

To contribute translations, visit:
https://translate.wordpress.org/projects/wp-plugins/lucide-acf-field-free/
